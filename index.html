Good Generator.py

Yet another static site generator. Written in Python. Why?

It powers its website, which is its README.md rendered via circleci. An infinite incepted loop!

It also powers my website.

Usage

  • Copy gg.py, ggconfig.py, Makefile and newpost.sh to the root of your website/blog
  • Adjust ggconfig.py to your preferences. Since it is an imported Python file, you can use it for preprocessing hooks, too!
  • Place markdown files everywhere
  • Install dependencies 
    make install_pipenv
make init
  • Generate your site 
    make
  • To update your gg.py "installation", run (your configuration will not be touched) 
    make update

Tests

  • Install dependencies (if you have not yet) 
    make install_pipenv
make init
  • Run the tests 
    make test

Features

Good Generator supports Markdown with various extensions!
